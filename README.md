- By the second half of 2017, being inspired by really cool and uplifting stuff on Youtube like [Crash Course Computer Science](https://www.youtube.com/watch?v=tpIctyqH29Q&list=PLH2l6uzC4UEW0s7-KewFLBC1D0l6XRfye), [CS50](https://www.youtube.com/watch?v=y62zj9ozPOM&list=PLhQjrBD2T3828ZVcVzEIhsHVgjANGZveu) and [this data structure series](https://www.youtube.com/watch?v=lC-yYCOnN8Q&list=PL2_aWCzGMAwI3W_JlcBbtYTwiQSsOTa6P&index=4), I started taking CS courses at University and consolidated my interest in the computing world. It was all so fun. It was like an addiction.

- In 2018 during an internship in my city, I learned about production-line challenges and heard the term "Industry 4.0" for the first time.

- I was convinced of the immense value waiting to be unlocked by companies lacking visibility of their processes, and I've been pursuing that vision ever since.
